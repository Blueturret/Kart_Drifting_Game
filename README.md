# Kart-Game
A Godot game about a shopping kart drifting in a supermarket as if the economy wasn't in a horrible spot right now.
This project was made for a devlog on my YouTube channel (French only) : https://www.youtube.com/@Blueturr3t

# How to play
The goal of the game is to get the highest score in a singleplayer race that's all about drifting between obstacles and tight turns. Losing momentum will decrease the score modifier, so the objective is to one-shot the whole track.

This repo is the whole Godot project that you can freely modify and/or from which you can reuse resources. The project was made in Godot 4.6, the assets are in glTF 2.0 format exported from Blender

The game (will be) available on itch.io. For now here's something to do while the game is being developped : http://www.republiquedesmangues.fr/
